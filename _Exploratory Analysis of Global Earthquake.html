# Standardize common column names: magnitude, depth_km, latitude, longitude, tsunami_flag
possible_mag = [c for c in df.columns if 'mag' in c.lower() or 'magnitude' in c.lower()]
possible_depth = [c for c in df.columns if 'depth' in c.lower()]
possible_lat = [c for c in df.columns if 'lat' in c.lower() or 'latitude' in c.lower()]
possible_lon = [c for c in df.columns if 'lon' in c.lower() or 'longitude' in c.lower()]
possible_tsu = [c for c in df.columns if 'tsunami' in c.lower()]

if possible_mag:
    df['magnitude'] = pd.to_numeric(df[possible_mag[0]], errors='coerce')
if possible_depth:
    df['depth_km'] = pd.to_numeric(df[possible_depth[0]], errors='coerce')
if possible_lat:
    df['latitude'] = pd.to_numeric(df[possible_lat[0]], errors='coerce')
if possible_lon:
    df['longitude'] = pd.to_numeric(df[possible_lon[0]], errors='coerce')
if possible_tsu:
    # normalize to 0/1
    df['tsunami_flag'] = df[possible_tsu[0]]
    try:
        df['tsunami_flag'] = pd.to_numeric(df['tsunami_flag'], errors='coerce').fillna(0).astype(int)
    except:
        df['tsunami_flag'] = df['tsunami_flag'].astype(str).str.lower().map({'yes':1,'y':1,'true':1,'1':1,'no':0,'n':0,'false':0}).fillna(0).astype(int)

# Show standardized columns
cols_to_show = [c for c in ['year','month','magnitude','depth_km','latitude','longitude','tsunami_flag'] if c in df.columns]
display(df[cols_to_show].head())
print("\nAfter standardization - missing values:\n", df[cols_to_show].isnull().sum())

Summary Reports:

Objective:

Analyze global earthquake and tsunami data (20012022) using Python to uncover patterns, trends, and seismic indicators associated with tsunami events.

Dataset Summary:- 
782 earthquake records (20012022)- No missing values- Key columns: magnitude, depth, latitude, longitude, Year, Month, tsunami

Key Findings:

Tsunami events are less frequent but cluster in specific years. Tsunami-generating earthquakes tend to have higher magnitudes (7.0) and shallower depths. Major earthquakes (8.0 magnitude) have a high probability of producing tsunamis. Geographical clusters appear along the Pacific Ring of Fire including Japan, Indonesia, Chile, and Alaska. Correlation analysis shows positive relation between magnitude and tsunami occurrence and negative relation with depth.

Major Earthquakes (8.0 Magnitude):

A total of 28 major events recorded, including:- 2021 (8.2M) Alaska Tsunami triggered- 2019 (8.0M) Peru Tsunami triggered- 2017 (8.2M) Mexico Tsunami triggered Conclusion: Tsunami-generating earthquakes are primarily characterized by high magnitude and shallow depth in coastal subduction zones. These insights can enhance early warning systems and risk assessments for vulnerable regions
